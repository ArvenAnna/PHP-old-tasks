Задача: форма для генерации списка дат
Форма имеет набор элементов:
1 ) 2 поля для ввода дат: "Дата с" и "по". Около каждого из них есть кнопка с
"календариком", при нажатии открывается календарь для выбора даты мышкой. Но
при этом можно указать даты и вводом с клавиатуры.
2) Раскрывающийся список с названиям дней недели ("Понедельник", "Вторник" ...
"Воскресенье")
3) Кнопку "Submit".
При нажатии на Submit генерируется список дат в рамках указанного диапазона
(включительно), но только те, которые соответствуют выбранному дню недели. Т.е.
если выбран понедельник, то выводятся все даты понедельников в указанном
диапазоне. Даты в этом списке выводятся в формате: “DD.MM.YYYY TS”, где TS –
timestamp начала суток по GMT.
В проблемных случаях должна выводиться понятная для пользователя ошибка.
Результат: zip архив с файлами, один из них index.php Если залить архив в
DOCUMENT ROOT вебсервера и открыть http://mydomain.com/index.php должна
работать эта форма.
